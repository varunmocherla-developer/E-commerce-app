# ShopWave Showcase

A modern e-commerce portfolio project built with React and Vite.

## Features

- Responsive landing page with premium storefront styling
- Search and category-based product filtering
- Wishlist toggles and cart preview panel
- Pricing summary with shipping and discount logic
- Portfolio-friendly structure that is easy to extend

## Run locally

```bash
npm install
npm run dev
```
