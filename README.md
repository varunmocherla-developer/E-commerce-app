# ShopWave E-commerce App

A clean and modern e-commerce web app built with React, JavaScript, HTML, and CSS. This project is designed to showcase a polished shopping experience with product discovery, cart interaction, wishlist support, and responsive design.

## Features

- Product listing with modern card layout
- Category-based filtering
- Search functionality
- Add to cart and remove from cart
- Wishlist toggle
- Order summary with subtotal, shipping, discount, and total
- Featured collection section
- Fully responsive design

## Tech Stack

- React
- JavaScript
- HTML5
- CSS3
- Vite
